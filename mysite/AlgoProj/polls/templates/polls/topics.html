Key Topics Covered This Semester - A General Overview

Divide and Conquer

Divide and Conquer is true to its name - recursively dividing the problem into sub problems, then working at them and building back up.

Recurrence Solving Techniques

This includes the Tree method (visualizing a tree and dividing to look at the overall picture); Guess and Check (using a mathematical guess, and checking using induction); "Cookbook"/Master Theorem (recurses on smaller problems, then generalizing to a form); Substitution (Changing the structure of the question, then returning to the original form and substituting)

Closest Pair of Points

Closest Pair of Points was, when given a number of points in a space, finding the pair of points that are the closest to each other, but using the smallest distance, delta, to calculate and examine the runway of 2delta.

Strassen's Algorithm

The idea behind Strassen's Algorithm is to multiply n by n matrices, such that it breaks down bigger matrices, increases the number of small subproblems, then builds back up. It is a fast algorithm for large matrices.

Sorting Algorithms

1. QuickSort - picks a pivot, then recursively sort each of the list to its 'left' and 'right'
2. QuickSelect - uses index i; picks a pivot and divides accordingly, then recursively operates on the divided lists using the index i
3. Randomized QuickSort - Randomized Quicksort takes quicksort, but chooses a random pivot. In doing so, a better runtime is sometimes achieved.
4. MSCS - Maximum Sum Continuous Subarray; The sum of the values of the elements in the array is maximal.
There are many more sorting techniques but for the sake of available real-estate on here, I've limited it to a few.

Dynamic Programming

Dynamic Programming is when the optimal solution of the overall problem is solved from the optimal solution of the smaller problem. There are two ways - top down, or bottom up.

Matrix Chain Multiplication

Ties under Dynamic Programming, but it aims to solve the optimal/most efficient way to multiply the matrices.

Seam Carving

Seam carving aims to scale the image, such that images with similar pixels are removed, around perhaps a focal object in an image. The pixels that are removed can be thought of as having "lowest energy", or least variance within that pixel.

Longest Common Subsequence

Just like the name, the longest common subsequence looks for the longest subsequence present, such that it finds the ones that fit the set criteria, and is the longest sequence of a given problem.

Greedy Algorithms

Greedy Algorithm looks at one solution, and solely makes the decision to follow that solution. It's not always the most optimal solution, which is why greedy algorithm may not be used all the time. However, when an algorithm chooses to follow one sub route of an algorithm and ignores any other, that makes the algorithm greedy.
